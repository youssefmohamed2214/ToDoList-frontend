## ğŸ“ Usage Guide

### Adding Tasks

1. Enter your task in the "Add a new task" field
2. Set the due date using the date picker
3. Select a priority level (High, Medium, Low)
4. Click "Add Task"

### Managing Tasks

- **Complete a task**: Click the checkbox next to a task
- **Edit a task**: Click the "Edit" button, make changes, and click "Save"
- **Delete a task**: Click the "Delete" button next to a task
- **Delete all tasks**: Click the "Delete All" button (requires confirmation)

### Organizing Tasks

- **Sort**: Click the up/down arrows in the column headers
- **Filter**: Use the "Filter By" dropdown to show all, completed, or uncompleted tasks
- **Search**: Enter text in the search box and select a search criteria
